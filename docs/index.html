How lots are generated
  • Roll 1D6 for column (trade class).
  • If the rolled class is not present on the world, treat it as None.
  • Apply any matching per-column modifiers (DMs).
  • Re-select the trade class after DM adjustment.
  • Roll 1D6 for row within the resulting column.
  • Select the merchandise entry whose roll range matches.
  • Apply Buy/Sell DMs and evaluate legality.
