Speculative Trade Generator
Loads trade tables from JSON and generates lots via 1D6 column + 1D6 row.

Mode

World trade codes

Select the world’s trade codes. Column rolls that don’t match a selected trade code fall back to None.

Generation

Advanced inputs

Advanced mode applies the columnDM rules from trade-columns.json to the column roll, and applies buy/sell DMs and simple legality checks (LG vs Law Level) on each generated item.
